# Support Guide

## Public help and bug reports

Use the [Q&A category](https://github.com/Antman316/agent-master-key-community/discussions/categories/q-a)
for non-sensitive setup questions and reproducible bugs. Include:

- Agent Master Key version;
- macOS version;
- agent and connector involved;
- steps to reproduce;
- expected and actual behavior; and
- confirmation that all evidence is redacted and contains no secrets.

## Private support

Email `support@agentmasterkey.com` for account-specific or private setup help. Do not include
credentials, payment data, raw logs, or unredacted screenshots.

## Security

Email `security@agentmasterkey.com` for vulnerabilities or suspected exposure. Do not post
security details publicly. See [SECURITY.md](SECURITY.md).

Maintainers may create a redacted internal engineering ticket from a confirmed report. The
private engineering board is not exposed through this repository.
