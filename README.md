# Agent Master Key Community

This public repository is the discussion and feedback hub for
[Agent Master Key](https://agentmasterkey.com), a local-first Mac app for giving AI agents
scoped, revocable access without placing raw provider credentials in the agent's hands.

The Agent Master Key source repository and engineering issue board are private. This
repository contains community documentation only; it does not contain product source code.

## Start here

- [Ask for help or report a non-sensitive bug](https://github.com/Antman316/agent-master-key-community/discussions/categories/q-a)
- [Suggest an idea](https://github.com/Antman316/agent-master-key-community/discussions/categories/ideas)
- [Read announcements](https://github.com/Antman316/agent-master-key-community/discussions/categories/announcements)
- [Download Agent Master Key](https://agentmasterkey.com/download)
- [Read the documentation](https://agentmasterkey.com/docs)

## Never post secrets

Discussions are public and searchable. Never post API keys, Agent Master Key keys, OAuth
codes, passwords, payment information, account email addresses, private local paths, raw
logs, or unredacted screenshots. Maintainers will remove exposed credentials and may lock
the discussion while directing you to the appropriate private channel.

- Private setup or account support: `support@agentmasterkey.com`
- Security vulnerabilities or suspected credential exposure: `security@agentmasterkey.com`

By participating, you agree to follow the [Code of Conduct](CODE_OF_CONDUCT.md),
[Support guide](SUPPORT.md), [Security policy](SECURITY.md), and [Community privacy notice](PRIVACY.md).
