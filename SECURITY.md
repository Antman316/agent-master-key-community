# Security Policy

Do not report vulnerabilities or suspected credential exposure in a public Discussion.

Email `security@agentmasterkey.com` with:

- a concise description of the issue;
- the affected Agent Master Key version and macOS version;
- safe reproduction steps using false or dummy credentials;
- the security impact you observed; and
- redacted screenshots or diagnostics only when needed.

Never send real API keys, Agent Master Key keys, OAuth codes, passwords, payment details,
raw provider payloads, or unredacted logs. We will respond as soon as the report has been
triaged; no response-time guarantee is made by this community repository.
